다음 예제를 통해 어떻게 실행 컨텍스트가 생성되고 실행 결과가 관리되는지, 그리고 어떻게 실행 컨텍스트를 통해 식별자를 검색하는지 살펴보자.

```javascript
var x = 1;
const y = 2;

function foo (a) {
	var x = 3;
	const y = 4;

	function bar (b) {
		const z = 5;
		console.log(a + b + x + y + z);
	}
	bar(10);
}

foo(20); // 42
```
